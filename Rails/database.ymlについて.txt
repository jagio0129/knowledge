Railsのアプリケーションを作成すると自動的に「datavase.yml」ファイルが作成されデフォルトの設定が記述される。
特に指定しない場合はSQLiteを使う前提で作成されるため、ほかのデータベースを使用する場合はRailsのアプリケーションを作成するときにオプションでデータベースをしてする。
	$ rails new [app_name] -d [db]
	
使用可能なデータベースは「mysql / oracle / postgresql / sqlite3 / frontbase / ibm_db / sqlserver / jdbcmysql / jdbcsqlite3 / jdbcpostgresql / jdbc」

datavase/ymlの中身
	adapter:   使用するデータベース種類
	encoding:  文字コード
	reconnect: 再接続するかどうか
	database:  データベース名
	pool:      コネクションプーリングで使用するコネクションの上限
	username:  ユーザー名
	password:  パスワード
	host:      MySQLが動作しているホスト名
